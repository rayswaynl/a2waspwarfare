# Agent Development Pack

This is a compatibility alias for prompts and agents that ask for an "agent development pack".

The canonical pack is [LLM agent entry pack](LLM-Agent-Entry-Pack). Use that page for the current AI boot order, safety rules, task bundles and output rules.

## Use This Instead

| Need | Canonical page |
| --- | --- |
| AI boot order | [LLM agent entry pack](LLM-Agent-Entry-Pack) |
| Compact machine bootstrap | [`agent-entrypoint.json`](agent-entrypoint.json) |
| Curated external LLM map | [`llms.txt`](llms.txt) |
| Current repo rules and page snapshot | [`agent-context.json`](agent-context.json) |
| Current lanes | [Progress dashboard](Progress-Dashboard), [`agent-status.json`](agent-status.json), [`agent-collaboration.json`](agent-collaboration.json) |
| Feature/risk triage | [Feature status register](Feature-Status-Register), [`agent-feature-status.jsonl`](agent-feature-status.jsonl) |

## Why this page exists

Some older prompts, scout notes and human shorthand say "agent development pack". Keeping this alias prevents future agents from creating another duplicated onboarding page.

Do not add new boot-order content here. Update [LLM agent entry pack](LLM-Agent-Entry-Pack) instead.
